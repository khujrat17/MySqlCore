# Contributing to MySqlCore

Thank you for considering contributing to MySqlCore! We welcome all contributions — bug reports, feature requests, and pull requests.

## How to Contribute

1. **Fork the repository**  
2. **Create a branch** for your feature or bugfix  
   ```bash
   git checkout -b feature/my-new-feature
Make your changes with clear, descriptive commits

Run tests to ensure everything works

dotnet test


Submit a Pull Request with a detailed description of your changes

Code Guidelines

Follow C# naming conventions (PascalCase for public members, camelCase for private variables)

Add XML documentation for public classes/methods

Write unit tests for new features or bug fixes

Keep PRs small and focused

Reporting Issues

Use the Issues tab to report bugs or request features

Provide repro steps, expected vs actual behavior, and environment details

Thank You!

Your contributions make MySqlCore better for everyone!
